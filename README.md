# Color-sorting-machine-using-Arduino
Automated Hardware-Software system designed to detect, identify, and sort physical objects, like M&amp;Ms, based on their color using an **Arduino Uno** and a **Color sensor TCS3200**
## Components
1. Microcontroller: Arduino UNO R3
2. Sensor: TCS3200
3. Actuators: 2 SG90 Servo Motors
4. Structure: 3D Cardboard Layout
5. Power: 4 batteries 1.5V each (6V in total)

The repository contains the source code for the Arduino, circuit diagram made in Fritzing App, CAD designing of the layout and the full documentation for the project in the pdf file **Color_sorting_machine_documentation**
